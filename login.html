Google Signin Authorization Demo

Note, to run this code you'll need to have your server running, either localhost on a PC or use terminal to run the following in the folder where you store these files: python -m SimpleHTTPServer 9000

For example my url is: http://localhost:9000/login.html

Click sign in to see examples of the user's profile data used on screen
